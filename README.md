# ejercicios SOLID

> En este repositorio, encontrarás cada letra de los principios SOLID representada por código Java que deberás refactorizar para cumplir correctamente con el principio correspondiente.

### Para completar esta tarea, deberás:

- Crear un nuevo repositorio a partir de este **repositorio de plantillas**.
- Navegar hasta las carpetas correspondientes a cada principio.
- Analizar el código problemático proporcionado.
- Refactorizarlo para que se ajuste al principio SOLID.
- Escribir una explicación de tus decisiones.

## Cómo empezar con el repositorio de plantillas

> En lugar de clonar este repositorio directamente, tendrás que crear tu propio repositorio a partir de este **repositorio de plantillas** para poder trabajar cómodamente y tener tu propio historial de cambios.

1. Haz clic en **Usar esta plantilla**.
2. Crea un nuevo repositorio con tu nombre y configuración.
3. A continuación, clona el nuevo repositorio que has creado en tu máquina local con:

```bash
   git clone https://github.com/usuari/nom-del-teu-repositori.git
   cd nombre-de-tu-repositorio
```

## Cómo completar la tarea

- Lee el archivo README.md de la carpeta del principio que quieras estudiar.
- Analiza la clase Java problemática proporcionada.
- Identifica cómo y por qué esta clase viola el principio SOLID.
- Refactoriza la clase para que se adhiera correctamente al principio.
- Reflexiona sobre los cambios:
1. ¿Qué estaba mal?
2. ¿Por qué violaba el principio?
3. ¿Qué solución has aplicado y por qué?
-  Escribe una breve explicación en un archivo llamado EXPLICACIO.md en la misma carpeta que el ejercicio.
